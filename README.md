### Problem Description
Please develop a web page featuring a web server routine that converts numerical input into words and
passes these words as a string output parameter.

### Approach

## Front-end
Create a web application using a C# framework. Razor has been implemented in this instance.
To run the application, navigate into the TechTest folder and run `dotnet run` from the terminal.
The web application is hosted at http://localhost:7258. 
This application features an input field in which a float (decimal) number can be input.
The form uses native validation.
